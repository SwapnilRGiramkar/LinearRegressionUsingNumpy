# Linear Regression Using Numpy Matplotlib
## Description
Developed a Machine Learning project that implements  Linear Regression from scratch using Python without relying on high-level ML libraries. The project demonstrates how a model learns the relationship between house size and price using the  Gradient Descent optimization algorithm.

Key aspects of the project include implementing the cost function (Mean Squared Error), training the model through multiple epochs, updating parameters (slope and intercept) using gradients, and generating predictions for new data inputs. The dataset is normalized to improve training performance and convergence.

The final results are visualized using Matplotlib, where the actual data points and the predicted regression line are plotted to clearly show how the model fits the dataset.

## Technologies Used:
Python, NumPy, Matplotlib

## Key Concepts:
Linear Regression, Gradient Descent, Cost Function, Model Training, Data Visualization
